#Notes:
-----------------------
Start server with:

npm run start

then visit http://localhost:8080


#Development:
-----------------------
Source code lives in the app directory. 


#Tests
-----------
  `karma start`
    

#Check for updates
-----------
  
  `npm-check-updates`


#Deploy:
-----------------------

  Make sure the gem is installed:
  
  `gem install s3_website`

  Build:

  `gulp build --release`
  
  Deploy:
  `s3_website push`
